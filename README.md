# CloudContacts (Agenda de Contactos en la Nube)

Instalación del entorno y creación de la estructura del proyecto Flask:
sudo apt update && sudo apt upgrade -y
sudo apt install python3-venv -y
python3 -m venv venv
source venv/bin/activate
pip install flask

Ahora crearemos la estructura de carpetas para el proyecto:
mkdir mi_proyecto
cd mi_proyecto
mkdir static templates
mkdir static/css static/js static/img
nano app.py
nano env.
nano requeriments.txt
